# Guide

## 1. Configuração do Ambiente de Desenvolvimento

- Instalar as ferramentas:
    - Editor de código;
    - NPM e Node;
    - Git;
- Criar repositório Git;
    - Definir .gitignore;
    - Criar branchs;

## 2. Setup do Backend;

- Iniciar projeto node;
- Instalar dependências:
    - Express;

## 3. Setup do Frontend;

- Usar npx create-react-app;
- Instalar dependências:
    - Redux;
    - React Router;
    - Axios;
